# FreeTVG [Tree View Gui]
## **Tree View Gui is an outline note for viewing in tree structure.**
### **Visit [TVG](https://treeviewgui.work)**
## Installation
```pip install FreeTVG-karjakak==1.0.1```
## Usage
```Python
import TVG

# Start TVG outline note
TVG.main()
```
